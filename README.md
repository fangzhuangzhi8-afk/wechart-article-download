WeChat Official Account Article Downloader

A tool to read and download full text and image content from WeChat Official Account articles and generate a complete Word document.

Core Features

- Complete Content Extraction: Extracts all text content from the article while preserving the original formatting.
- Batch Image Download: Downloads all images and supports various image formats.
- Position Preservation: Places images in their correct positions within the text, mirroring the original article layout.
- Word Generation: Creates a well-formatted Word document in .docx format.

Technical Features

- Supports both single URL and batch processing.
- Automatically identifies the HTML structure of WeChat Official Account articles.
- Comprehensive error handling and progress feedback.
- Generates processing reports and metadata files.
